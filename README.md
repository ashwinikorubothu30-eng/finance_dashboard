# Finance Dashboard

A responsive and interactive Finance Dashboard built using React.

## 🚀 Features
- Dashboard overview (Total Balance, Income, Expenses)
- Charts (Monthly trends & Category breakdown)
- Transactions table with search and filter
- Role-based UI (Admin / Viewer)
- Insights section
- Dark mode support
- Currency formatted in Indian Rupees (₹)

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts

## ▶️ How to Run
npm install
npm run dev
